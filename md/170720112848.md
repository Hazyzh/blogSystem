### a标签的'download'属性

*html5* 中a标签增加了download选项，`<a href="/images/logo.jpg" download="logo">`
设置了download就可以当作资源下载，而不是出现文件预览图。

```javascript
	function download(name, url) {
		var s = document.createElement("a")
		s.download = name
		s.href=url
		s.click()
	}
```
